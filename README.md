AI Personal Sports Trainer Prompt

English Description

This project contains a prompt design for an AI-powered personal sports trainer built with LangChain and OpenAI GPT-3.5.
It includes structured prompting techniques such as Zero-Shot, Chain-of-Thought, ReAct, and Role Prompting, allowing the AI to create personalized workout and nutrition plans for different users and goals.

Prompt Goals
• Create structured, personalized fitness plans.
• Adapt to different training goals (bulking, cutting, maintenance, lifestyle).
• Provide motivational and logical explanations for every response.
